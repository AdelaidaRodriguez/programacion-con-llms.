## Descripción del Proyecto
Este repositorio contiene una serie de ejercicios diseñados para practicar la manipulación de datos y el preprocesamiento para Machine Learning, utilizando herramientas de Inteligencia Artificial de forma ética y productiva.

## Estructura de la Fase 1
En la carpeta `myquestions` encontrarás 4 retos técnicos:

* **Pregunta 0001 (Pandas):** Limpieza de valores nulos en columnas específicas.
* **Pregunta 0002 (Scikit-Learn):** Imputación de datos faltantes usando la mediana.
* **Pregunta 0003 (Pandas):** Filtrado condicional complejo para gestión de inventarios.
* **Pregunta 0004 (Scikit-Learn):** Estandarización de escalas con StandardScaler.

Cada reto incluye su respectivo **Generador de casos de uso** (`-usecase-generator.py`) para validar automáticamente las soluciones propuestas.
